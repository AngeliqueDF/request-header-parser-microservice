Request header parser microservice

This microservice responds to a GET request at the http://localhost:5000/api/whoami endpoint with the following information about the client:

  • IP address
  • The browser language
  • The software used by the client (browser, operating sytem)

For example, the following information was found on this browser:

IP address: test
Language: test
Software: test
